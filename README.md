# 🚀 NeonStrike Studios - Web Interface Project

Bu proje, modern web teknolojileri ve yapay zeka araçları entegre edilerek hazırlanmış bir oyun stüdyosu tanıtım arayüzüdür.

## 🛠️ Kullanılan Teknolojiler
* **Frontend:** HTML5, CSS3, Bootstrap 5
* **AI Görsel:** Leonardo AI
* **AI Seslendirme:** ElevenLabs (Jediah Voice)
* **IDE:** Cursor AI
